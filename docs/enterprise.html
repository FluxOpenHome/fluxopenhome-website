How It Works

Connect any Flux irrigation controller to the management dashboard in three steps.

1

Install the Controller

Set up the Flux Irrigation Controller at the property. Works with the Home Assistant Add-on or Direct Connect firmware.

2

Generate a Connection Key

The homeowner generates a connection key from their HA Add-on. Or use Direct Connect for properties without Home Assistant.

3

Manage from Your Dashboard

Add the property to your dashboard via key or QR scan. Full remote monitoring and control from day one.

Direct Connect

No Home Assistant Required

For commercial installs where simplicity matters, the Direct Connect version of the Flux Irrigation Controller connects straight to the cloud management dashboard via WiFi.

  • Same ESP32-S3 hardware, different firmware
  • WebSocket connection to the management server
  • Full zone control, scheduling, and weather rules
  • Ideal for new construction and commercial properties
  • Same open hardware design — firmware source available

Our Open Source Commitment

The management dashboard is a convenience layer, not a control gate.

Your Data Stays Local

The Home Assistant Add-on keeps all data on the homeowner's hardware. The management dashboard only sees what they explicitly share via connection key.

No Vendor Lock-in

All Flux hardware runs open firmware. If you leave the management platform, your controllers keep working exactly as before.

The HA Add-on Is Always Free

Homeowners never pay for local control. The Home Assistant Add-on with all its features is free and open source, permanently.
