Specifications

Built on the ESP32-S3 with full ESPHome integration and expansion capability.

MCU ESP32-S3 DevKitC-1
Zones 8 built-in (GPIO45, 35-41), expandable to 32
Expansion MCP23017 I2C boards (addresses 0x20-0x22)
Zone 8 Configurable as normal zone, master valve, or pump start relay
Rain Sensor GPIO4, configurable NO/NC, 1-72 hour delay
Status LED WS2812 on GPIO42
Scheduling 4 start times, auto-advance, repeat cycles
Valve Overlap 3-second seamless transition between zones
WiFi Provisioning Bluetooth, captive portal, serial
OTA Updates Yes
Firmware v1.2.7

Features

Everything you need for intelligent, reliable irrigation management.

Weather-Based Irrigation

9 configurable weather rules via the Home Assistant Add-on. Automatically adjust watering schedules based on forecasts, temperature, and rainfall data.

Rain Sensor Support

Normally Open or Normally Closed sensor types with configurable rain delay from 1 to 72 hours. Automatic irrigation shutoff on rain detection.

Expansion Boards

Scale up to 32 zones with MCP23017 I2C expansion boards. Auto-detection of connected boards at addresses 0x20 through 0x22.

Smart Scheduling

4 programmable start times with 7-day scheduling. Auto-advance cycles through enabled zones automatically with repeat cycle support.

Pump / Master Valve

Zone 8 is configurable as a pump start relay with automatic 2-second startup delay. Stays on throughout the entire irrigation cycle.

OTA Firmware Updates

Update wirelessly from the ESPHome dashboard or Home Assistant. Firmware releases hosted on GitHub with automatic update notifications.

Home Assistant Add-on

Unlock advanced irrigation management with the Flux Irrigation Add-on for Home Assistant.

The Flux Irrigation Home Assistant Add-on (v1.2.2) extends your controller with a full management dashboard and intelligent automation features. Install directly from the Home Assistant Add-on Store or from GitHub.

  • Homeowner dashboard with zone status and scheduling overview
  • Weather-based irrigation with 9 configurable rules
  • Gophr soil moisture probe integration for data-driven watering
  • Rain sensor management and delay configuration
  • PDF system reports for irrigation history and diagnostics
  • Connection keys for lawn care and management companies
Install Add-on

The Add-on runs locally on your Home Assistant server. All data stays on your network. No cloud account required for the Home Assistant version.

Management companies can connect remotely using connection keys generated from the homeowner dashboard, enabling professional monitoring without requiring direct access to your Home Assistant instance.
