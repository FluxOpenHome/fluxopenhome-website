Frequently Asked Questions

Yes! We provide STL files for the Night Light housing. You can buy just the circuit board from our store and print the housing yourself. The files are optimized for FDM printers and require no supports.

The base unit supports 8 zones. With expansion boards (MCP23017 I2C), you can expand up to 32 zones total. Zone 8 can also be configured as a master valve or pump start relay.

Gophr comes in three versions: Home Assistant ($100) which connects via ESPHome WiFi for full local control, Direct Connect ($100) which connects via WiFi to the Flux Cloud using MQTT with no Home Assistant required, and Cellular ($150) which has a built-in cellular modem for remote locations beyond WiFi range.

Absolutely not. The management dashboard is entirely optional. All Flux products work independently with Home Assistant. The dashboard is a convenience layer for professionals who manage multiple properties — it does not gate any hardware functionality.

All Flux devices support OTA (Over-the-Air) updates. When connected to Home Assistant, updates can be pushed from the ESPHome dashboard. The Night Light also has a built-in web server for direct browser-based updates, and GitHub-hosted firmware releases.
