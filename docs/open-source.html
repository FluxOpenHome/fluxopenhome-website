Home Assistant Integration

All Flux products are designed for seamless Home Assistant integration. Install the Flux Irrigation Add-on for advanced features:

  • Weather-based smart irrigation with 9 configurable rules
  • Gophr moisture probe monitoring and automated watering decisions
  • Homeowner dashboard with zone scheduling and control
  • Rain sensor management with configurable delay (1-72 hours)
  • PDF system reports for water usage tracking
  • Connection key sharing for management company access
